# Assets do Projeto Surprise Manoela

## Estrutura de Pastas

```
public/assets/
├── photos/          # Fotos para o feed Instagram
├── restaurants/     # Imagens dos restaurantes Kharina
├── audio/          # Arquivos de áudio
└── fonts/          # Fontes customizadas (se necessário)
```

## Arquivos Necessários

### 📸 Fotos (public/assets/photos/)
Adicione as fotos do relacionamento com os seguintes nomes:
- `photo1.png`
- `photo2.png`
- `photo3.png`
- `photo4.png`

> Se as fotos não forem encontradas, o sistema usará imagens placeholder do Unsplash.

### 🍽️ Restaurantes (public/assets/restaurants/)
Imagens dos restaurantes Kharina:
- `kharina-batel.jpg`
- `kharina-cabral.jpg`

> Se não encontradas, serão exibidos ícones de placeholder.

### 🎵 Áudio (public/assets/audio/)
Arquivo de áudio da dubladora da Barbie:
- `barbie_audio.mp3` - Áudio que toca na cena final

> **Importante**: O áudio deve ser colocado nesta pasta para funcionar na cena final do quiz.

## Como Adicionar Assets

1. Crie as pastas se não existirem
2. Copie os arquivos para as pastas correspondentes
3. Reinicie o servidor Next.js se estiver rodando

## Dicas

- Use imagens otimizadas (WebP preferencialmente)
- Tamanho recomendado para fotos: 800x800px
- Tamanho recomendado para restaurantes: 600x400px
- Áudio MP3 em boa qualidade (128kbps+)

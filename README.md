## 🤖 Guia para IA (Claude/ChatGPT/Antigravity)

Para que um assistente de IA possa trabalhar neste projeto de forma eficiente, siga estas diretrizes:

1. **Contexto**: Este repositório contém projetos interativos de aniversário (como `birthday-surprise` e `surprise-manoela`).
2. **Tecnologias**: 
   - `birthday-surprise`: HTML/JS/CSS vanilla com integração de áudio (Spotify/YouTube).
   - `surprise-manoela`: Next.js 14+, Tailwind CSS, TypeScript.
3. **Padrões de Commit**: Use o padrão de Conventional Commits (ex: `feat:`, `fix:`, `chore:`).

## 🚀 Fluxo de Trabalho Git

Sempre que realizar alterações, utilize os comandos abaixo:

### 1. Adicionar alterações
```bash
git add .
```

### 2. Criar um Commit
```bash
git commit -m "sua mensagem descritiva aqui"
```

### 3. Enviar para o GitHub
```bash
git push origin main
```

## 🛠️ Como rodar os projetos localmente

### Birthday Surprise (Vanilla)
Basta abrir o arquivo `index.html` em um navegador ou usar um servidor local simples:
```bash
npx serve birthday-surprise
```

### Surprise Manoela (Next.js)
```bash
cd surprise-manoela
npm install
npm run dev
```
